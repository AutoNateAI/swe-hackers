## How This Works

1. **Cards/Issues** = Assignments with tutorials, challenges, and resources
2. **Project Board** = Track what's in progress, done, or blocked
3. **Your Folder** = Where you do the actual work

Browse the [**Project Board**](https://github.com/users/nathanbaker-ao/projects/1) to see your cards.

## Quick Start

```bash
# Clone the repo (if you haven't)
git clone https://github.com/nathanbaker-ao/swe-hackers.git
cd swe-hackers

# Pull latest changes
git pull origin main

# Go to your folder
cd brian  # or marcus

# Create a project folder for your assignment
mkdir my-project
cd my-project

# Do your work, then commit and push
git add .
git commit -m "Complete assignment X"
git push origin main
```
